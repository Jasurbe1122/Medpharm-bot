Telegram Botni Ishga Tushirish:
1. Python 3.8+ o‘rnatilgan bo‘lsin.
2. `pip install -r requirements.txt` bilan kutubxonalarni o‘rnating.
3. BotFather’dan token oling va `.env` fayliga `TELEGRAM_BOT_TOKEN=your_token` sifatida kiriting.
4. `python bot.py` bilan botni ishga tushiring.
5. Guruhda `/start` va `/guruhhisobot` buyruqlarini sinab ko‘ring.
6. Tokenni hech qachon ommaviy ulashmang.
Bot Pydroid, lokal server yoki Render’da ishlaydi.
